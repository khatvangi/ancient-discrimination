SEQUENCE-BASED DNA vs RNA BINDING DISCRIMINATION TOOLS — ASSESSMENT
====================================================================
date: 2026-02-24
purpose: find tools that classify proteins as DNA-binding vs RNA-binding from sequence alone

TOOLS SURVEYED
--------------

1. DRBP-EDP (Mu et al. 2025, NAR Genomics & Bioinformatics)
   - INSTALLED AND WORKING
   - location: /storage/kiran-stuff/DRBP-EDP/
   - conda env: drbp-edp
   - method: ESM-2 (650M) + dual-path neural network (Transformer+CNN & BiLSTM+Attention)
   - two-stage: (1) NABP vs non-NABP, (2) DBP vs RBP
   - protein-level classification
   - GitHub: https://github.com/MuQiang-MQ/DRBP-EDP

2. ProNA2020 (Qiu et al. 2020, J Mol Biol, Rostlab)
   - INSTALLED AND WORKING
   - location: /storage/kiran-stuff/ProNA2020/
   - conda env: prona2020
   - method: word2vec (ProtVec) + gradient boosting classifiers
   - gives separate DNA/RNA/Protein binding probabilities
   - protein-level classification (word2vec pathway only; full version needs PredictProtein)
   - GitHub: https://github.com/Rostlab/ProNA2020

3. DRNApred (Yan & Kurgan 2017, NAR)
   - WEB-SERVER ONLY (behind access-approval wall)
   - residue-level predictor (DNA-binding residues vs RNA-binding residues)
   - no standalone code available
   - server: https://biomine.cs.vcu.edu/servers/DRNApred/

4. DRBpred (Kabir et al. 2024, Computers in Biology and Medicine)
   - CLONED but NOT PRACTICAL for our use
   - residue-level predictor (not protein-level)
   - requires Docker + nr database + uniclust30 database (very heavy)
   - location: /storage/kiran-stuff/DRBpred/
   - GitHub: https://github.com/wasicse/DRBpred

5. DeepDRBP-2L (Liu lab)
   - WEB-SERVER ONLY (http://bliulab.net/DeepDRBP-2L — server unresponsive)
   - protein-level: classifies DBP, RBP, and DRBP (dual-binding)
   - no standalone code available

6. bindPredict / bindEmbed21 (Rostlab)
   - Available on GitHub but predicts "nucleic acid" binding generically
   - does NOT discriminate DNA vs RNA at the binding level
   - GitHub: https://github.com/Rostlab/bindPredict

7. SomeNA (Rostlab, 2012)
   - Available via Docker
   - requires PredictProtein output files as input (complex pipeline)
   - unclear whether it discriminates DNA vs RNA separately
   - not practical for direct sequence input

8. GPSite (biomed-AI, 2024, eLife)
   - residue-level predictor (DNA/RNA/protein/peptide/ATP/metal)
   - would need post-processing to get protein-level scores
   - requires ESMFold + ProtTrans (heavy install)
   - GitHub: https://github.com/biomed-AI/GPSite

RECOMMENDATION FOR THE PROJECT
-------------------------------

Use BOTH tools together. They have complementary strengths:

1. DRBP-EDP: better stage1 (is it a nucleic acid binder?), ESM-2 based, modern
2. ProNA2020: gives separate DNA and RNA probabilities (not forced choice),
   better at TP53-like DNA-binding proteins

Neither tool alone is reliable enough. For our ancient discrimination project:
- Run both tools on each protein
- Report the "discrimination index" as the difference between DNA and RNA scores
- ProNA2020's separate DNA_prob and RNA_prob are more directly useful for our
  question ("did ancient proteins discriminate?") than DRBP-EDP's forced binary

Key limitation: both tools are trained on modern proteins, so applying them to
ancestrally reconstructed sequences may be pushing beyond their training domain.
This is fine — we acknowledge it. Structural prediction (AF3/Boltz-2) is
planned as a complementary method (Phase 3, not yet completed), with
sequence-based tools as additional cross-checks alongside the primary
NAS-Bench structural contact analysis.
